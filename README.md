# Bank Account

#### Mock bank account webapp

#### Carl Egbert and Shradha Pulla

## Description

Simple mock bank account webpage that allows users to create accounts and make withdrawals and deposits.

## Specs

The program should handle: | Example Input | Example Output
----- | ----- | -----
An initial deposit. | John Smith, $5,000,000,000 | John Smith, $5,000,000,000
A deposit. | +$1 | $5,000,000,001
A withdrawal. | -$2 | $4,999,999,999

## Technologies Used

* Bootstrap
* JQuery
